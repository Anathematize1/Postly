# Postly
Веб-приложение для публикации постов и ведения личных блогов. Пользователи могут создавать записи, комментировать публикации и взаимодействовать с контентом других пользователей. 


## 📌 Описание проекта: Postly — это социальное веб-приложение, в котором реализована функциональность публикации постов, работы с комментариями и управления пользовательским контентом. 
Пользователь может вести собственный блог, просматривать записи других пользователей, оставлять комментарии и управлять своими публикациями. 


## 🚀 Функциональность: 

### 👤 Пользователи: 
- регистрация и авторизация пользователей
- просмотр профиля пользователя
- разграничение прав доступа


### 📝 Посты:
- создание публикаций
- редактирование и удаление своих постов
- просмотр постов других пользователей
- отображение постов по категориям и авторам
- поддержка изображений в публикациях


### 💬 Комментарии: 
- добавление комментариев к постам
- редактирование и удаление своих комментариев
- отображение комментариев под постами


### 🛠 Админ-панель: 
- управление пользователями, постами и комментариями
- настройка отображения моделей
- поиск и фильтрация данных
-  расширенная конфигурация админки


## 🛠 Стек технологий: 
- Python 
- Django
- SQLite
- HTML, CSS
- Bootstrap


## ⚙️ Запуск проекта: 1. Клонировать репозиторий:
bash```
git clone <repo_url>
cd blogicum```
2. Создать виртуальное окружение:
bash```
python -m venv venv
source venv/bin/activate  # Linux / Mac
venv\Scripts\activate     # Windows```
3. Установить зависимости:
bash```
pip install -r requirements.txt```
4. Применить миграции:
bash```
python manage.py migrate```
5. Запустить сервер:
bash```
python manage.py runserver```
6. Открыть в браузере:
bash```
http://127.0.0.1:8000/```


##📌 Особенности проекта: - реализована система прав доступа (пользователь управляет только своими объектами) - используется Django ORM для работы с базой данных - реализована пагинация - настроена расширенная админ-панель - спользуется шаблонизация Django - реализована работа с формами и валидацией данных
